## 기능
* 유저 로그인 / 로그아웃 (JWT 사용) + 회원가입
    * 로그인 : email
    * 비밀번호 : 8글자 이상 + 특수문자 + 숫자 포함
* 매 주별 해야할 일의 등록 / 수정 / 삭제 / 조회 기능
* 일정이 지난 후 진행한 할 일들에 대한 해결여부 기록
* routine_day 테이블에서는 복합키 사용
* 테스트 코드 작성
